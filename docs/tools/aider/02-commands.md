# 2. 命令详解（42 个）

Aider 提供 42 个斜杠命令，按功能分为以下类别。

## 文件与上下文管理

| 命令 | 用途 |
|------|------|
| `/add <file>` | 添加文件到对话上下文 |
| `/read-only <file>` | 以只读方式添加文件 |
| `/drop <file>` | 从上下文移除文件 |
| `/ls` | 列出上下文中的文件 |
| `/map` | 显示仓库地图 |
| `/map-refresh` | 刷新仓库地图 |
| `/context` | 显示上下文信息 |
| `/tokens` | 显示 token 用量统计 |

## 模式切换

| 命令 | 用途 |
|------|------|
| `/chat-mode <mode>` | 切换聊天模式 |
| `/code` | 切换到代码编辑模式 |
| `/architect` | 切换到架构师模式（先规划后编辑） |
| `/ask` | 提问模式（不编辑代码） |

## 模型管理

| 命令 | 用途 |
|------|------|
| `/model <name>` | 切换主模型 |
| `/editor-model <name>` | 设置编辑器模型 |
| `/weak-model <name>` | 设置弱模型（用于提交消息等） |
| `/models <query>` | 搜索可用模型 |

## Git 与工作流

| 命令 | 用途 |
|------|------|
| `/commit [msg]` | 提交当前更改 |
| `/undo` | 撤销上次 Git 提交 |
| `/diff` | 显示最近的 diff |
| `/git <cmd>` | 执行 Git 命令 |
| `/lint` | 运行代码检查器 |
| `/test <cmd>` | 运行测试命令 |
| `/run <cmd>` | 执行 Shell 命令 |
| `/web <url>` | 抓取网页内容加入上下文 |

## 会话管理

| 命令 | 用途 |
|------|------|
| `/clear` | 清除对话历史 |
| `/reset` | 重置上下文（移除所有文件 + 清除历史） |
| `/settings` | 显示当前设置 |
| `/load <file>` | 从文件加载命令 |
| `/save <file>` | 保存命令到文件 |

## 输入输出

| 命令 | 用途 |
|------|------|
| `/paste` | 粘贴剪贴板内容 |
| `/copy` | 复制上次回复 |
| `/voice` | 语音输入 |
| `/editor` | 在外部编辑器中编辑 |
| `/edit <file>` | 编辑指定文件 |
| `/multiline-mode` | 切换多行输入模式 |

## 其他

| 命令 | 用途 |
|------|------|
| `/help` | 显示帮助信息 |
| `/ok` | 无消息继续执行 |
| `/report` | 报告问题 |
| `/think-tokens` | 控制思维 token 预算 |
| `/reasoning-effort` | 设置推理努力级别 |
| `/copy-context` | 复制上下文信息 |
| `/exit` | 退出 Aider |
| `/quit` | 退出 Aider |
